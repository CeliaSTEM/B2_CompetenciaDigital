Creamos una web segura, accesible y útil para nuestra comunidad educativa

Reto para el alumnado

¿Qué reto vamos a resolver?

La comunidad educativa necesita recursos claros y comprensibles para utilizar internet de forma segura y responsable. Por ello, cada equipo deberá crear una página web que ayude a otros estudiantes a proteger sus datos, respetar la privacidad de los demás, usar recursos con licencias adecuadas y publicar contenidos digitales de forma responsable.

Texto

¿Cómo podemos crear una web que ayude a otros estudiantes a cuidar su identidad digital y navegar por internet de forma segura?

  1. Elegir la temática.
  2. Buscar información fiable.
  3. Revisar licencias y protección de datos.
  4. Diseñar la estructura de la web.
  5. Crear la web.
  6. Revisar accesibilidad y seguridad.
  7. Presentar el producto final.
  8. Mejorarlo a partir del feedback recibio
