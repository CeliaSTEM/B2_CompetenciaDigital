Producto final

Nuestro producto final: una web segura y accesible

El producto final será una página web informativa creada por equipos. La web estará dirigida a otros estudiantes del centro y deberá ofrecer recomendaciones útiles sobre seguridad digital, protección de datos, identidad digital, licencias y accesibilidad.

La web deberá incluir:

  • Título claro.
  • Menú o estructura organizada.
  • Información fiable y bien redactada.
  • Imágenes o recursos citados correctamente.
  • Recomendaciones sobre seguridad digital.
  • Ausencia de datos personales.
  • Diseño visual claro.
  • Criterios básicos de accesibilidad.
  • Enlaces revisados.
  • Presentación final ante el grupo.

Herramientas recomendadas:

  • Visual Studio Code.
  • HTML y CSS.
  • Navegador web.
  • Google Drive.
  • Google Sites como alternativa guiada.
  • Padlet.
  • Miro.
