Plan ante incidencias técnicas

¿Qué hacemos si surge un problema técnico?

Durante el proyecto pueden aparecer dificultades técnicas: problemas de conexión, errores de código, pérdida de archivos, enlaces rotos o dificultades para utilizar alguna herramienta digital. Estas incidencias se abordarán como oportunidades para aprender a resolver problemas de forma progresiva y autónoma.

Protocolo de ayuda:

  • Intento identificar el error.
  • Consulto la guía de errores frecuentes.
  • Pregunto a mi equipo.
  • Pregunto a otro equipo.
  • Solicito ayuda al profesorado.
  • Registro la solución para poder reutilizarla.

Plan B: 

  • Guardar copias en Drive.
  • Trabajar offline si falla internet.
  • Usar Google Sites si el desarrollo con código resulta demasiado complejo.
  • Revisar el código por parejas.
  • Utilizar plantillas comentadas.
  • Registrar incidencias y soluciones.
