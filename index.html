Creamos una web segura, accesible y útil para nuestra comunidad educativa

Inicio

En esta Situación de Aprendizaje vamos a diseñar y crear una página web informativa dirigida a la comunidad educativa. El objetivo será elaborar un recurso digital claro, seguro, accesible y útil sobre identidad digital, protección de datos, licencias, seguridad en internet y uso responsable de la tecnología.

Trabajaremos de forma cooperativa, utilizando herramientas digitales para buscar información, organizar contenidos, crear una web, revisar su funcionamiento y evaluar el proceso seguido.

Datos básicos:

Nivel: CFGM de Sistemas Microinformáticos y Redes.
Módulos: Aplicaciones Web y Seguridad Informática.
Temporalización: 9 sesiones.
Producto final: página web informativa.
Autoría: Celia Pedregosa.
